# Security Policy

Pock Community is an unofficial community-maintained fork of Pock. Please do not report security-sensitive issues in public GitHub issues.

## Reporting A Vulnerability

Open a private security advisory on GitHub if available for this repository, or contact the repository owner through GitHub with a short non-sensitive summary and a request for a private channel.

Do not include secrets, Apple Developer certificates, provisioning profiles, private crash dumps, access tokens, or personal data in public issues.

## What To Include

- Affected Pock Community version or commit.
- macOS version and Mac model.
- Whether the Mac has a physical Touch Bar.
- Clear reproduction steps.
- Impact assessment.
- Logs with private information removed.

## Scope

Security reports may include:

- Local privilege escalation risks.
- Unsafe handling of downloaded widgets or archives.
- Code execution risks in widget loading or installation.
- Privacy issues caused by unexpected data collection or transmission.

This fork does not include telemetry, analytics, or tracking.

## Response Expectations

Maintainers should acknowledge valid reports as soon as practical, avoid public disclosure until a fix is available, and credit reporters when requested and appropriate.
